### Mixins

将组件中都能用到的代码提出 用Mixins 混入到组件中



```js
--组件.js
{
  mixins:[fetchData],
  data(){
    return{
      
    }
  },
  methods:{
    
  }
}
```

```js
-- fetchData.js
	混入的created先执行
export default{
  data(){
    return{
      
    }
  },
  created(){
    
  },
  methods:{
    
  }
}
```

```js
Vue.mixins({

  }
})
```
