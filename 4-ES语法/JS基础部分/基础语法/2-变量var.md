## 变量

#### 什么是变量

>  变量是一块内存空间，用于保存数据

标识符规范：

1）只能以英文字母、下划线、$开头

2）其他位置可以出现数字、英文字母、下划线、$

3）不能与关键字、保留字重复

4）⚠️注意语义化

5）使用小驼峰方式命名

语法糖：

```js
单一 var变量

var a,
    b, 
    c;
var a = 2,
    b = 3,
    c = 4;

```

### 变量声明提升

```js
console.log(a,b) //
var a = '123',
var b;
b = 2;
```
