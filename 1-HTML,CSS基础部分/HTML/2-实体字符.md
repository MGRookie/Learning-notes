### 实体字符

| 显示结果 | 描述     | 实体名称 | 实体编号      |
| -------- | -------- | -------- | :------------ |
|          | 空格     | &nbsp;   |               |
| <        | 小于号   | &lt;     | &lt;          |
| >        | 大于号   | &gt;     | `>`           |
| &        | 和号     | &amp;    | `&`           |
| "        | 引号     | &quot;   | `"`           |
| '        | 撇号     | &apos;   | `'`(IE不支持) |
| ￠       | 分       | &cent;   | `¢`           |
| ￡       | 镑       | &pound;  | `£`           |
| ￥       | 日元     | &yen;    | `¥`           |
| €        | 欧元     | &euro;   | `€`           |
| §        | 小节     | &sect;   | `§`           |
| ©        | 版权     | &copy;   | `©`           |
| ®        | 注册商标 | &reg;    | `®`           |
| ™        | 商标     | &trade;  | `™`           |
| ×        | 乘号     | &times;  | `×`           |
| ÷        | 除号     | &divide; | `÷`           |
