# 多媒体元素


1.video 视频
2.audio 音频

## video

```html
    <video src='mp4/webm' controls autoplay muted loop ><video>
```

controls： 播放控件的显示，取值只能controls
autoplay：表示自动播放  
muted：静音播放
loop：循环播放

## audio

和视频一样


> 兼容性 mp4 和 webm 有的浏览器 会有兼容性的问题
用如下方法解决：
```html
<video>
    <source src='xxx.mp4'>
    <source src='xxx.webm'>
    <p>对不起你的浏览器不支持播放此视频！！请下载Chrome浏览器 </p>
</video>
```
