#### css权重

| 选择器            | 权重     |
| :---------------- | -------- |
| !impotent         | infinity |
| 行间样式          | 1000     |
| id                | 0100     |
| class、属性、伪类 | 0010     |
| 标签、伪元素      | 0001     |
| 通配符*           | 0000     |

`256进制`
