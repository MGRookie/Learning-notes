# 常见样式声明

1. font-size
   元素内部文字的尺寸大小

1）px：像素，绝对单位，简单的理解为文字的高度占多少个像素
2）em：相对单位，相对于父元素的字体大小
每个元素必须有字体大小，如果没有声明，则直接使用父元素的字体大小，如果没有父元素（html），则使用基准字号。

2. font-family:
   必须用户计算机中存在的字体才会生效
   sans-serif，非衬线字体

3. font-style：
   italic //斜体
   normal

4. text-indet：
   首行文本缩进

5. text-decoration：
   文字修饰线

a 元素
del 元素：错误的内容，中间划线
s 元素：过期的内容

6. line-height：
   px em 1.5

7) letter-space：
   文字间隙

8) textarea
   文本输入框
