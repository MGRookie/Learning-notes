### css选择器

##### id选择器

```html
<div id='id1'>
demo
</div>
```

```css
#id{
  
}
```



##### class选择器

```html
<div class='class1'>
demo
</div>
```

```css
.class1{
  
}
```



##### 标签选择器

```html
<div>
  demo
</div>
```

```css
div{
  
}
```



##### * 通配符选择器

```html
<div>
  demo
</div>
```



##### 属性选择器

> 用方括号选择属性，只要是有这个属性的标签就会被选中，可以适用于自定义属性名

```html
<div id='id1' class='class1' e5e6e7e8e8e >
  demo
</div>
```

```css
#id1[e5e6e7e8e8e]{
  
}

或
[id='id1']{
  
}

或
[class]{
  
}
```

#### 复合选择器

##### 派生选择器｜后代选择器

```css
div a{

}

- div下的所有a元素
```

##### 直接子元素选择器

```css 
div > a{

}

- div的子元素a
```
