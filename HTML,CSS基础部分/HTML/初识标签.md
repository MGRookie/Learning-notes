## html

html  ：Hyper Text Markup Language  HTML的全称为超文本标记语言，是一种[标记语言](https://baike.baidu.com/item/HTML/97049?fr=aladdin)。

#### lang  属性

> 表示该网页的语言

```html
<html lang='en,zh'>
  
</html>
```

#### meta 标签

> 功能标签 定义在**head** 里面

```html
<head>
  <meta charset="utf-8"
</head>
```

#### title

> 标题标签 定义浏览器标签标题

```html
<head>
  <title> 淘宝网 ！ 淘你喜欢！</title>
</head>
```
