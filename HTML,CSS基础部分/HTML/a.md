#### a

> 超链接



```html
<a href='' target='_blank' rel="noopener noreferrer"></a>
```

> 锚点

```html
<a href='#idname'>跳转到某个元素</a>
```

> 打电话

```html
<a href = 'tel:16655021802'></a>
```

> 发邮件

```html
<a href = 'mailto:574169815@qq.com'></a>
```

> 协议限定符

```html
<a href = 'javascript:viod(0)'
```
